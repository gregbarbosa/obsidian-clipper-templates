To install templates, see instructions in the official [Obsidian Web Clipper documentation](https://help.obsidian.md/web-clipper/templates) repo.

# Philosophy

My vault is not a "read it later" app. My vault is a shrine to how I think and connect to what I've learned. Every clipped piece of content is read, enjoyed, and referenced as backlinks in other notes. 

## Clipping

My default template used when something sparks a new idea or connects to another concept I've been working on. Immediately after the piece is clipped, I'll reference it within the other note using backlinks.

## Company

Used as reference for companies I want to keep notes on. A barebones pseudo-CRM, where I fill out diary-like entries.

## Website

My catch-all template for websites that continue to spark joy and inspire me.

## Fujifilm Recipe

Using Interpreter, this template extracts a Fujifilm recipe from a page to build out your own recipes library. Add a reference image as the source to create a beautiful Fujifilm Recipe Base view.
